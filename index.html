Setup Google Sheets

Ikuti langkah berikut untuk menghubungkan aplikasi ke Google Sheets kamu.

1
Buka Google Sheets baru, klik Extensions → Apps Script
2
Hapus semua kode yang ada, lalu paste isi file Code.gs yang sudah disiapkan, klik Save
3
Klik Deploy → New Deployment, pilih type Web App, set Execute as: Me dan Who has access: Anyone, klik Deploy
4
